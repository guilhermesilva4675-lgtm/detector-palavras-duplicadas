# Detector de Palavras Duplicadas

Projeto desenvolvido durante os estudos de Node.js.

## Funcionalidades

- Lê arquivos .txt
- Identifica palavras duplicadas por parágrafo
- Gera um arquivo resultado.txt
- Interface de linha de comando (CLI)

## Tecnologias

- Node.js
- Commander
- Chalk

## Como executar

Instale as dependências:

npm install

Execute:

node src/cli.js -t arquivos/texto-web.txt -d ./resultados
